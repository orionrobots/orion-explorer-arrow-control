orion-explorer-arrow-control
============================

This is a set of Arduino and Processing sketches to drive an Orionrobots Explorer 1 Robot with your keyboard on a PC.
You can read the full tutorial at [orionrobots.co.uk/explorer_arrow_control](<https://orionrobots.co.uk/explorer_arrow_control.html>)

This repository is maintained at [orionrobots/orion-explorer-arrow-control](<https://github.com/orionrobots/orion-explorer-arrow-control>)
